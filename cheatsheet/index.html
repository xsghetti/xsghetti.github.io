CSS Cheatsheet

CSS Selectors & Specificity

If there are two or more CSS rules that point to the same element, the selector with the highest specificity value will "win", and its style declaration will be applied to that HTML element.

Selectors Name Description Specificity
#UniqueHeader ID Selection Selects a unique element with an ID attribute 1
.header Class Selection Selects elements with a specific class attribute. 2
p, h1, div Type Selection Selects HTML elements based on the element name. 3

CSS Display Properties

Property Description
Inline The inline display value turns any element into an inline element. These elements will appear on the same line without breaks, like <span> elements behave.
Block The block display value makes an element a block element. Block elements start a new line and span the entire width of the viewport by default, like how <div> elements behave. There are also line breaks before and after these elements.
Inline Block The inline-block value is a hybrid of inline and block. Elements assigned display: inline-block appear on the same line with other inline elements, a characteristic of inline elements. However, inline-block elements are also like block elements in that you can change their widths and heights with CSS.
